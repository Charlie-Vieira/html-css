Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B não semântica.

Nesta frase temos um termo em destaque usando a tag STRONG.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I não semântica.

Nessa frase temos um termo em ênfase usando a tag EM.

Texto Marcado

Podemos criar também um Texto Marcado usando a tag MARK.

Podemos colorir o Texto Marcado usando CSS em forma de Mark Style.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno utilizando as tags BIG e SMALL, mas a tag BIG não é mais utilizada profissionalmente.

Texto deletado

Podemos marcar um texto como deletado para indicar que ele deve ser lido, mas não considerado, utilizando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois, utilizando a tag INS.

Existe também o texto sublinhado, que utiliza a tag U, mas não é semântica.

Texto Sobrescrito

Para inserir coisas do tipo x10+3, utilizando a tag SUP nas letras superiores.

Texto Subescrito

Para inserir coisas do tipo H20, utilizando a tag SUB nas letras inferiores.
